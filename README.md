# Maryam Ghaffar – Portfolio Website

This is my personal portfolio website. It shows who I am, what I do, and the projects I’ve built.

## 🔗 Live Website
[Click here to view my portfolio](https://maryamghaffar1234m-png.github.io/Maryam_Portfolio/)  


## 📁 Files in this project
- `index.html` – main structure of the website  
- `style.css` – all colors, layout, animations  
- `script.js` – interactive parts (projects, certificate popup, contact form)  
- `Maryam_CV.pdf` – my resume (downloadable)  
- `certificate.png` – my Generative AI certificate image  

## ✨ Features
- Clean and responsive design (works on mobile, tablet, laptop)
- Smooth scrolling and fade-in animations
- Hover effects on buttons, skills, and project cards
- Working contact form (opens your email app – no backend needed)
- Certificate image popup on click
- Downloadable CV (PDF)
- Projects section – easy to update via JavaScript array

## 🛠️ Technologies used
- HTML5
- CSS3 (Flexbox, Grid, animations, gradients)
- JavaScript (ES6)
- Font Awesome icons
- Google Fonts (Inter)

## 🚀 How to run locally
1. Download all files into one folder.
2. Open `index.html` in any browser.
3. That’s it – no server or installation needed.

## 🌐 Deployment
This website is hosted on **GitHub Pages**.  
Any changes pushed to the `main` branch will automatically update the live site.
